## Installation & Usage

Download project dependencies 

```console
uv sync
```

Activate venv (Windows)
```console
.venv\Scripts\activate 
```

Activate venv (Linux)
```console
source .venv/bin/activate
```

Start local webserver
```console
make start-local
```
